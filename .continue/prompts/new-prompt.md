---
name: commit-essage
description: Generate concise Git commit messages based on provided diffs or contexts.
invokable: true
---

1. Execute o comando 'git diff' para obter todas as alterações não comitadas.
2. Analise o resultado do comando 'git diff' para identificar as alterações realizadas.
3. Com base nas alterações identificadas, gere uma mensagem de commit concisa e descritiva.
4. Responda com uma mensagem para o commit, seguindo as os requisitos:
   1. A mensagem deve ser em português.
   2. A mensagem deve ser objetiva e descritiva.
   3. Inclua na mensagem uma lista das alterações realizadas, separadas por vírgula.
   4. A mensagem deve seguir as convenções de commit do projeto.
   5. A mensagem deve ser em uma única linha.
5. Não inclua nenhuma outra informação na resposta além da mensagem de commit.
